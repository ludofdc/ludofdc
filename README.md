## Hi there 👋

I work on computational models for physical systems, with a focus on building, solving and validating numerical simulations from first principles.

My main project is a research-oriented thesis on the simulation of electron spill-out from thin metallic slabs, combining density functional theory, partial differential equations and numerical methods.

---

### Main work
**Simulation of electron spill-out from a thin metallic slab**

- Self-consistent numerical modeling based on DFT
- Discretization and solution of coupled PDEs
- Core solvers implemented in C++
- Data analysis and visualization in Python
- Emphasis on numerical stability, convergence and physical consistency
- Here you can find the link to the repo: https://github.com/ludofdc/electron-spillout-slab
